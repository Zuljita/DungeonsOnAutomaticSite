Compact status or map-marker label; tones mirror the map-key colors so a badge reads the same as the symbol on the map.

```jsx
<Badge tone="encounter" dot>Encounter</Badge>
<Badge tone="treasure">+200% loot</Badge>
<Badge tone="ok">Validated</Badge>
<Badge tone="key" solid>Key 3</Badge>
```

Tones: `neutral | encounter | treasure | hazard | key | door | secret | ok | warn | danger`. `solid` for high emphasis, `dot` for a leading status dot.
