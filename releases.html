Continuous early access builds

These unofficial early access builds are mirrored as the continuous public release. They are usable desktop builds, not a hosted web generator.

The Windows and Linux artifacts are large because of desktop packaging and bundled app dependencies. The local LLM runtime is not bundled; install Ollama, LM Studio, or llama.cpp separately if you want AI Narrative Polish.

New here? Start with the first dungeon quickstart, then return for downloads and known issues.

Windows installerRecommended Windows x64 setup build. About 1.43 GB. macOS Apple siliconZIP build for M-series Macs. About 263 MB. macOS IntelZIP build for Intel Macs. About 151 MB. Linux x86_64AppImage build for 64-bit Linux. About 1.79 GB. Linux ARM64AppImage build for ARM64 Linux. About 1.88 GB.

Known issues
Current roadmap

Current limitations

  • Single-level generation only; stairs are placed and exported, but true multi-floor graphing is future work.
  • Universal VTT .dd2vtt output exists with embedded map image, walls, portals, lights, and GM/player modes, but importer hardening is ongoing.
  • Custom treasure per room and custom container tables are still partial.
  • Variable hallway widths are not implemented yet.
